# API JAVASCRIPT - PROTOTYPE

Application web développée en JavaScript permettant d’interagir avec des APIs et d’afficher les données de manière dynamique et interactive.

## 🚀 Fonctionnalités

- Consommation d’API : Récupération et affichage des données depuis une API.
- Interface dynamique : Mise à jour du contenu en temps réel avec JavaScript.
- Design responsive : Compatible avec mobile et desktop.
- Gestion des erreurs : Affichage des messages d’erreur en cas de problème API.

## 🛠️ Stack Technique

- Frontend : HTML, CSS, JavaScript
- API : Fetch API
- Outils : Git, VS Code
